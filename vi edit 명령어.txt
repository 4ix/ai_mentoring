제가 진짜 사용하는 vi 명령어들 알짜 모음> 명령모드 (esc 눌렀을때, vi 처음 들어갔을때)
1. 파일의 끝으로 이동할때는 -1G
2. 한줄 잘라내기 -1dd
3. 세줄 잘라내기 - 3dd
5. 한줄복사 - 1yy
6. 두줄복사 - 2yy
4. 붙여넣기 -1p
5. 한글자 삭제 - x
6. 단어 삭제 - dw
7. 실행취소! - u
8. 줄의 맨 앞 - o
9. 줄의 맨 뒤 - $(G, $ 이거 개꿀)
> 마지막행 모드 (esc -> : 눌렀을때)(명령어 뒤에 !를 붙이면 강제로 수행하니까, vi가 궁시렁 궁시렁 거리면" ! "붙여서 실행 하면 됩니다.)

2. 종료만 : q
3. 저장 후?종료 : wq
